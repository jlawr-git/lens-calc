Import / export

JSON export preserves both formats and lenses in one file. CSV export produces two files since CSV is a flat format — re-importing a CSV will only update whichever type matches its columns. Imports merge by name (same-name entries get overwritten); they do not delete entries absent from the file. Reset to defaults clears both formats and lenses.

Equivalent focal length matches horizontal field of view — source focal × (target effective width ÷ source effective width), where effective width = sensor width × anamorphic squeeze (1.0 for spherical). Crop factor referenced against full-frame 36×24mm (43.27mm diagonal). Coverage check compares lens image circle diameter to target sensor diagonal — a difference under 2mm is flagged as marginal. Anamorphic image circles are often elliptical in reality; this tool treats them as circular for simplicity.
